# Tree_Trekker

A lightweight, robust Command-Line Interface (CLI) application built with Python. This tool allows users to seamlessly perform core file system management actions—such as creating, reading, editing, and deleting files or directories—without ever needing a graphical user interface (GUI).

---

## ✨ Features

- **File Operations:** Create new files safely, append content to existing files, read data inputs, and delete files with robust error handling.
- **Directory Management:** Create new folders, navigate forward or backward through your system directories, and view clean directory lists.
- **Safe Execution:** Built-in safeguards against `FileNotFoundError` and `FileExistsError` to ensure your data is never unintentionally overwritten.
- **Zero Dependencies:** Runs purely using Python's native standard library (`os`).

---

## 🚀 How to Run the App

### 📋 Prerequisites
- Ensure you have **Python 3.x** installed on your computer.

### ⚙️ Setup and Execution
1. **Download the project** directly from this GitHub page to your machine.
2. Open your computer's terminal, command prompt, or terminal window.
3. Use the `cd` command to enter the directory containing `project_Tree_Trekkar.ipynb`:
   ```bash
   cd path/to/your/downloaded-folder
   ```
4. Run the application using the python command:
   ```bash
   python project_Tree_Trekkar.ipynb
   ```
